Changelog
=========

v0.5.7
------
- Profile popup: Show height in meters (m) by default on Y axis; add "Z Layer" toggle button to switch to layer index
- Profile popup: Button label is axis-aware ("X Layer", "Y Layer", "Z Layer") when viewing X/Y/Z slices
- Profile popup: Y-axis tick labels use non-scientific notation (%.6g)
- Line profile popup (click-to-profile): X/Y/Z axes now default to physical coordinates (m); add "Layer" toggle button
- Map mode X/Y slices: Fix discontinuous color bands by computing dot size from actual screen-space cell size
- Map mode X/Y slices: Remove coastline overlay (map lines only shown for Z-slice)
- Map mode X/Y slices: Fix axis labels — Y-axis now shows "Height (m)"; X-axis shows "Longitude (deg)" for Y-slice and "Latitude (deg)" for X-slice

v0.5.6
------
- FFT popup: Add method toggle between 2D FFT and Wiener-Khinchin (Ma et al. 2024)
- FFT popup: Use centered square crop for 2D FFT (spectracam-style)
- FFT spectrum: Add three reference slopes with spectracam colors:
  - Purple k^(-1) Batchelor viscous-convective
  - Orange k^(-5/3) Kolmogorov inertial range
  - Pink k^(-3) Kraichnan 2D enstrophy cascade
- FFT popup: More square aspect ratio, method notes with automatic text wrapping
- Wiener-Khinchin mode: Shows horizontal (cyan) and vertical (green) spectra for isotropy check

v0.5.5
------
- SDM/SBM Settings: Add X-axis limits (X-min, X-max) in Settings dialog
- SDM mode: Add PDF toggle for probability density function normalization
- SDM/SBM info labels: Display total sum of all bins (e.g., total mass, total count)

v0.5.4
------
- Add FFT button: 2D energy spectrum popup with Kolmogorov k^(-5/3) reference line
- FFT method: Cooley-Tukey 2D FFT, Hann windowed, mean subtracted, power-of-2 truncation per axis
- FFT popup shows method notes describing normalization and wavenumber convention

v0.5.3
------
- SDM mode: X-axis now starts from 0 in linear mode

v0.5.2
------
- SDM mode: Add LogBin toggle for log-spaced bins (constant width in log space)
- SDM mode: Fix histogram bar display on log X axis using explicit bin edges
- SDM Settings: Fix click-to-focus on text input fields (Cutoff, Bin width)

v0.5.1
------
- Stats display: Add mean value alongside min/max at bottom-left of plot
- Distribution popup: Auto-update histogram when layer, variable, axis, or level changes

v0.5.0
------
- SBM mode: Add --sbm flag for Spectral Bin Microphysics droplet size distribution
- SBM metrics: QC Mass, QI Mass, Total Mass, QC Num, QI Num, Total Num
- SBM log-spaced bins: Use geometric mean for bin edges (sqrt of adjacent centers)
- SBM PDF mode: Normalize histogram to probability density function (integral = 1)
- Distribution popup: Add Layer/Domain mode toggle buttons
- Distribution popup: Show AMR level explicitly in title
- Improved Y-axis tick formatting for small values (PDF mode)

v0.4.3
------
- Combined Colorbar dialog: merge Colormap and Range buttons into single Colorbar button
- Log scale support: Linear, Log(+) for positive values, Log(-) for negative values
- Log scale grays out invalid values (non-positive for Log(+), non-negative for Log(-))
- Colorbar tick labels adapt to show 10^x format in log modes

v0.4.2
------
- Map dialog: Remove button now turns Map off and closes the window
- Map missing message includes PLTVIEW_WITH_MAP=1 install hint
- Rotate Y-axis label 90° CCW with improved spacing

v0.4.1
------
- Add optional map layer installation via pip and environment variable
- Map layers auto-discovery from installed package

v0.4.0
------
- Zoom and pan: mouse wheel zoom centered on cursor, Z+/Z- buttons, drag-to-pan when zoomed
- Axis labels update to show visible sub-range when zoomed
- Keyboard shortcut: 'r' or '0' to reset zoom
- Zoom resets automatically on variable, axis, level, or timestep change
- Map mode: dot size scales with zoom level for continuous coverage
- Map mode overlay: higher AMR levels now render data (not just boundaries) in map mode
- Click-to-profile works at all zoom levels (click = profile, drag = pan)
